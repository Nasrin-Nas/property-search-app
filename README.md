# Propert Search React App

This is a client-side web application that allows users to search properties by type, price, bedrooms, date added, and postcode area.  

- Built with **React JS** and **Vite**  
- Uses **JSON** data for property listings (no server)  
- Features search filters, results display, and favourites list  
- Responsive design for desktop and mobile
